#### Notation

Sets of fully-connected parties are denoted with `{}`, for example the
following shows a simple network of four parties:

<script type="text/tikz">
\begin{tikzpicture}
  \tikzstyle{every node}=[draw,circle]
  \node (A) at (0,1) {A};
  \node (B) at (1,1) {B};
  \node (C) at (0,0) {C};
  \node (D) at (1,0) {D};
  \path (A) edge (B);
  \path (A) edge (C);
  \path (A) edge (D);
  \path (C) edge (B);
  \path (B) edge (D);
  \path (C) edge (D);
\end{tikzpicture}
</script>

```
P = { A, B, C, D }
```

We can also nest sets of parties within each other, for example here's the same
network as above:

```
P = { { A, B }, { C, D } }
or
Q = { A, B }
R = { C, D }
P = { Q, R }
```

We can also define graphs that are not fully-connected with `[]`. For example a
"star" network with four edge nodes and a single center node can be represented
like this:

<script type="text/tikz">
\begin{tikzpicture}
  \tikzstyle{every node}=[draw,circle]
  \node (S) at (1,1) {S};
  \node (A) at (1,2) {A};
  \node (B) at (2,1) {B};
  \node (C) at (1,0) {C};
  \node (D) at (0,1) {D};
  \path (S) edge (A);
  \path (S) edge (B);
  \path (S) edge (C);
  \path (S) edge (D);
\end{tikzpicture}
</script>

```
P = { S, [A, B, C, D] }
or
P = [ { S, A }, { S, B }, { S, C }, { S, D } ]
```
